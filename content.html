การใช้งาน MQTT บน native ESP8266 + Arduino IDE และทดสอบเชื่อมกับ cloudmqtt

ตัวบอร์ด ESP 8266 ถือว่าเป็นบอร์ดที่ได้รับความนิยมอย่างแพร่หลาย เพราะเนื่องด้วยมีราคาถูก และความสามารถที่มันทำได้ และในครั้งนี้ เราจะลองนำตัว ESP8266 มาเชื่อกับ cloudmqtt ดูนะครับ และผมยังทำตัวอย่างที่ใช้ ESP8266 มาเก็บค่าอุณภูมิ และความชื้นจาก sensor DHT22 แล้วยิงขึ้นไปที่ Thingspeak เพื่อ เก็บ Log เอาไว้ดูผ่านเว็บได้ด้วย ตัวอย่าง

- ขั้นแรก เราไปดูที่ Code ที่เราจะเขียนลงใน ESP 8266 นะครับ

  • บรรทัดที่ 11 ssid คือชื่อของ Wifi ที่เราจะต่อ (ตัวใหญ่-ตัวเล็ก-ช่องว่างพิมพ์ให้ถูกนะ)
  • บรรทัดที่ 12 pass ก็คือ รหัสของ Wifi ตัวที่เราต้องการจะต่อ
  • บรรทัดที่ 15 IPAddress ก็ให้เราใส่ IP ของ server ของ MQTT ที่เราต้องการจะต่อเข้าไป (Port default คือ 1883)
  • บรรทัดที่ 23 ตรงนี้ถ้าใครต้องการเปลี่ยน Port ไปเป็น Port อื่นให้เปลี่ยนที่บรรทัดนี้ คือให้เติม Port ไปหลัง server ได้เลย เช่น

    PubSubClient client(server ,12345);

  • บรรทัดที่ 48 เป็นเหมือน chanel ที่ส่งข้อมูล
  • บรรทัดที่ 49 คือ Topic ที่เราต้องการจะ Subscribe
ตัวอย่างการทำ Log อุณภูมิบน Internet
